# Flights App

Java Version

```11```

Build Tool

```maven```

Build

```mvn clean package```

Run

```java -jar flights-0.0.1-SNAPSHOT.jar --flight.data=/path/to/flights.csv```


Search

``` http://localhost:8080/api/flights/2022/06/13 ``` 
``` http://localhost:8080/api/flights/year/month/day ```
